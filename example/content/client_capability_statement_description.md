This capability statement describes the expected capabilities of the US Quality
Core Client which is responsible for initiating queries for USCDI+
Quality V1 data from US Quality Core Servers.  The set of FHIR RESTful
operations and search parameters required to be supported by US Quality Core
Servers is provided in the [US Quality Core Server Capability
Statement](CapabilityStatement-us-quality-core-server.html).  US Quality Core
Clients have the option of choosing from this list to access necessary data
based on their local use cases and other contextual requirements.
